# Modul Quiz TJKT 2026

Modul pembelajaran interaktif TJKT berbasis React dan TypeScript.

## Fitur

- Dark Mode
- Quiz Interaktif
- Sound Effect
- Responsive
- Materi TJKT
- Progress Belajar

## Teknologi

- React
- TypeScript
- Vite
- Lucide React

## Menjalankan

npm install
npm run dev
